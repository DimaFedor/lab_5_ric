## To‑Do App

Простий React-додаток для ведення списку справ з збереженням у `localStorage`.

### Функціонал

- **Додавання задачі**: поле вводу + кнопка «Додати» (або клавіша Enter).
- **Позначення виконаних задач**: клік по тексту задачі.
- **Видалення задачі**: кнопка з хрестиком навпроти задачі.
- **Очищення всіх задач**: кнопка «Очистити все», якщо список не порожній.
- **Збереження стану**: усі задачі зберігаються в `localStorage` браузера.

### Структура проєкту (основне)

- `src/App.js` – кореневий компонент, який рендерить `TodoApp`.
- `src/components/TodoApp.jsx` – головний компонент додатку зі стейтом.
- `src/components/TodoInput.jsx` – поле вводу та кнопка додавання.
- `src/components/TodoList.jsx` – список задач.
- `src/components/TodoItem.jsx` – окремий елемент задачі.
- `src/App.css` – базові стилі.

### Як запустити

1. Встановити залежності:
   ```bash
   npm install
   ```
2. Запустити додаток у режимі розробки:
   ```bash
   npm start
   ```
3. Відкрити в браузері `http://localhost:3000`.

### Збірка для продакшну

```bash
npm run build
```

Готовий білд буде створено в папці `build/`.
